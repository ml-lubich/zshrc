## Requirements

- **Supported operating systems**
  - macOS (Apple Silicon and Intel)
  - Linux with one of: `apt`, `dnf`, or `yum`

- **Network**
  - Outbound HTTPS access to:
    - GitHub (for Homebrew, Powerlevel10k, and fonts)
    - Package repositories for your OS

- **Tools the script expects**
  - `curl`
  - `git` (installed automatically on Linux if a supported package manager is found)
